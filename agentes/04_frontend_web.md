# Agente Frontend Web - V1

## Missão

Entregar interface PT-BR para operação técnica diária, com foco em velocidade de consulta e clareza.

## Telas mínimas

- Login.
- Dashboard com indicadores e alertas.
- Lista de SaaS/APP com busca e filtros.
- Detalhe do SaaS/APP com abas de hospedagem, domínio, integrações, assinaturas e anexos.
- Tela de auditoria com filtros básicos.

## UX obrigatória

- Navegação em poucos cliques para informação crítica.
- Máscara visual de dados sensíveis por padrão.
- Fluxo explícito para revelar segredo (com confirmação).
- Mensagens e validações em PT-BR.

## Estados de interface

- Loading claro.
- Erro com ação de retry.
- Empty state com orientação de próximo passo.

## Critérios de aceite

- Busca retorna resultados corretos por campos principais.
- Cadastro completo do ativo técnico sem fricção excessiva.
- Alertas visíveis e acionáveis.
